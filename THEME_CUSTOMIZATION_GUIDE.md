# 🎨 دليل تخصيص الثيمات - Theme Customization Guide

## نظرة عامة

نظام تخصيص الثيمات المتقدم يسمح لكل مستخدم بتخصيص جميع ألوان الثيم بشكل كامل مع معاينة فورية ومباشرة للتغييرات.

---

## 🌟 المميزات الرئيسية

### ✨ تخصيص شامل
- **الخلفيات**: خلفية رئيسية، سطح، بطاقات، مرتفعة، جداول، فلاتر، حقول
- **Header & Footer**: خلفيات ونصوص Navbar و Footer
- **الألوان الأساسية**: أساسي، ثانوي، إبراز
- **النصوص**: ألوان النصوص الأساسية والثانوية والثالثية
- **الحالات**: نجاح، تحذير، خطأ، معلومات
- **الحدود والأيقونات**: حدود، فواصل، أيقونات
- **الروابط**: ألوان الروابط العادية وعند التمرير

### 🎯 معاينة فورية
- **تحديث مباشر**: كل تغيير يظهر فوراً على الشاشة
- **لا حاجة لإعادة تحميل**: التغييرات تطبق مباشرة عبر JavaScript
- **معاينة كاملة**: Navbar، جداول، بطاقات، أزرار، نصوص

### 💾 إدارة متقدمة
- **حفظ التخصيصات**: حفظ جميع التغييرات في قاعدة البيانات
- **إعادة تعيين**: إعادة جميع الألوان للقيم الافتراضية
- **تصدير**: تصدير التخصيصات كملف JSON
- **استيراد**: استيراد تخصيصات سابقة (قريباً)

## 🎨 كيفية الاستخدام

### 1️⃣ اختيار الثيم الأساسي
في أعلى اللوحة اليسرى، اختر الثيم الأساسي الذي تريد التخصيص منه:
- 🏠 الكلاسيكي (Default)
- 🌟 مخصص (Custom)
- 🌙 أسود عصري (Modern Black)
- 🍫 موكا موس (Mocha Mousse)
- 🌍 أرض دافئة (Warm Earth)
- ☕ أناقة القهوة (Coffee Elegance)

### 2️⃣ تخصيص الألوان
استخدم منتقيات الألوان لتغيير أي عنصر:

#### 🎨 الخلفيات
```
خلفية رئيسية     → لون خلفية التطبيق بالكامل
سطح              → خلفية الأسطح والعناصر
بطاقات          → خلفية البطاقات
مرتفعة          → خلفية العناصر المرتفعة
جداول           → خلفية الجداول
فلاتر           → خلفية نماذج البحث والفلاتر
حقول            → خلفية حقول الإدخال
```

#### 🎯 Header & Footer
```
خلفية Navbar     → لون خلفية شريط التنقل
نص Navbar        → لون النصوص في شريط التنقل
خلفية Footer     → لون خلفية التذييل
نص Footer        → لون النصوص في التذييل
```

#### 🎨 الألوان الأساسية
```
لون أساسي       → الأزرار الرئيسية والعناصر المهمة
لون ثانوي       → الأزرار والعناصر الثانوية
لون إبراز       → عناصر التمييز والتحديد
```

#### ✍️ ألوان النصوص
```
نص أساسي        → النصوص الرئيسية
نص ثانوي        → النصوص الأقل أهمية
نص ثالثي        → النصوص المساعدة والتلميحات
```

#### 🎭 ألوان الحالة
```
نجاح ✅         → رسائل النجاح والعمليات الناجحة
تحذير ⚠️       → رسائل التحذير والتنبيه
خطأ ❌          → رسائل الخطأ والمشاكل
معلومات ℹ️     → رسائل المعلومات العامة
```

#### 📐 الحدود والأيقونات
```
حدود            → لون الحدود حول العناصر
فواصل           → خطوط الفصل بين العناصر
أيقونات         → لون الأيقونات
أيقونات (تمرير) → لون الأيقونات عند التمرير عليها
```

#### 🔗 الروابط
```
روابط           → لون الروابط العادية
روابط (تمرير)   → لون الروابط عند التمرير عليها
```

### 3️⃣ المعاينة الفورية
في اللوحة اليمنى، ستشاهد معاينة فورية تتضمن:
- ✅ **Navbar**: شريط التنقل بالألوان الجديدة
- 📋 **جدول بيانات**: عينة جدول لمعاينة الجداول
- 🎴 **بطاقات**: عينات بطاقات بالتصميم الجديد
- 🔘 **أزرار**: جميع أنواع الأزرار
- 📝 **نصوص وروابط**: عينات النصوص والروابط

كل تغيير تقوم به سيظهر **فوراً** في المعاينة!

### 4️⃣ حفظ التخصيصات
عندما تنتهي من التخصيص:

#### 💾 حفظ
```
انقر "💾 حفظ التخصيصات"
← يتم حفظ جميع التغييرات في قاعدة البيانات
← التخصيصات ستطبق تلقائياً في المرة القادمة
```

#### 🔄 إعادة تعيين
```
انقر "🔄 إعادة تعيين"
← تأكيد العملية
← جميع التخصيصات تعود للقيم الافتراضية
```

#### 📤 تصدير
```
انقر "📤 تصدير"
← يتم تحميل ملف JSON يحتوي على جميع التخصيصات
← يمكن استخدامه لاحقاً للاستيراد
```

## 🔧 التفاصيل التقنية

### Model
```python
ThemeCustomization
├── user (OneToOne → User)
├── base_theme (CharField)
├── background_color (CharField)
├── surface_color (CharField)
├── ... (28+ حقل للألوان)
├── is_active (BooleanField)
├── created_at (DateTimeField)
└── updated_at (DateTimeField)
```

### Views
```python
theme_customization_view      → صفحة التخصيص الرئيسية
theme_customization_reset     → إعادة تعيين التخصيصات
theme_customization_export    → تصدير التخصيصات
theme_customization_update_live → تحديث مباشر (AJAX)
```

### URLs
```
/accounts/theme-customization/         → الصفحة الرئيسية
/accounts/theme-customization/reset/   → إعادة تعيين
/accounts/theme-customization/export/  → تصدير
/accounts/theme-customization/update-live/ → تحديث مباشر
```

### JavaScript
```javascript
// تحديث فوري للمتغيرات CSS
document.documentElement.style.setProperty('--background', color);

// خريطة ربط بين الحقول ومتغيرات CSS
cssVarMap = {
    'background_color': '--background',
    'primary_color': '--primary',
    ...
}
```

## 📊 أمثلة الاستخدام

### مثال 1: تخصيص ثيم أزرق
```
1. اختر "الكلاسيكي" كثيم أساسي
2. غيّر اللون الأساسي → #1976D2 (أزرق)
3. غيّر اللون الثانوي → #42A5F5 (أزرق فاتح)
4. غيّر خلفية Navbar → #0D47A1 (أزرق داكن)
5. احفظ التخصيصات
```

### مثال 2: ثيم داكن مخصص
```
1. اختر "أسود عصري" كثيم أساسي
2. غيّر الخلفية الرئيسية → #1A1A1A (أسود رمادي)
3. غيّر خلفية البطاقات → #2D2D2D (رمادي داكن)
4. غيّر النص الأساسي → #FFFFFF (أبيض)
5. غيّر اللون الأساسي → #00E676 (أخضر نيون)
6. احفظ التخصيصات
```

### مثال 3: ثيم دافئ
```
1. اختر "أناقة القهوة" كثيم أساسي
2. غيّر اللون الأساسي → #D84315 (برتقالي محروق)
3. غيّر خلفية الجداول → #FBE9E7 (وردي فاتح)
4. غيّر النص الأساسي → #3E2723 (بني داكن)
5. احفظ التخصيصات
```

## 🎯 نصائح مهمة

### ✅ نصائح للتخصيص الجيد

1. **التباين**: تأكد من وجود تباين جيد بين النصوص والخلفيات
   ```
   ❌ نص أبيض (#FFFFFF) على خلفية رمادية فاتحة (#F5F5F5)
   ✅ نص بني داكن (#3E2723) على خلفية رمادية فاتحة (#F5F5F5)
   ```

2. **الانسجام**: استخدم ألوان متناسقة من نفس العائلة
   ```
   ✅ أزرق داكن (#1976D2) + أزرق فاتح (#42A5F5) + أزرق باهت (#E3F2FD)
   ❌ أحمر (#F44336) + أخضر نيون (#00E676) + أصفر (#FFEB3B)
   ```

3. **القراءة**: اختبر قراءة النصوص على الخلفيات المختلفة
   ```
   - جرّب قراءة جدول كبير
   - تأكد من وضوح النصوص الصغيرة
   - اختبر في الإضاءة المختلفة
   ```

4. **الاحترافية**: لا تبالغ في استخدام الألوان الزاهية
   ```
   ✅ استخدم 2-3 ألوان أساسية فقط
   ❌ استخدام 10 ألوان مختلفة تماماً
   ```

### ⚠️ تحذيرات

- **لا تحفظ بدون اختبار**: تأكد من وضوح جميع العناصر قبل الحفظ
- **احتفظ بنسخة احتياطية**: صدّر تخصيصاتك قبل التغييرات الكبيرة
- **التباين للأزرار**: تأكد من وضوح الأزرار على جميع الخلفيات
- **الأيقونات**: اختبر ظهور الأيقونات بوضوح

## 🔍 استكشاف الأخطاء

### المشكلة: التغييرات لا تظهر
**الحل**:
1. حدّث الصفحة (F5)
2. امسح ذاكرة التخزين المؤقت (Ctrl+Shift+R)
3. تحقق من حفظ التخصيصات

### المشكلة: الألوان غير واضحة
**الحل**:
1. استخدم "إعادة تعيين" للعودة للقيم الافتراضية
2. اختر ثيم أساسي آخر
3. زد التباين بين النصوص والخلفيات

### المشكلة: المعاينة مختلفة عن الصفحات الفعلية
**الحل**:
1. احفظ التخصيصات أولاً
2. انتقل لصفحة أخرى لرؤية التطبيق الكامل
3. قد تحتاج بعض الصفحات لإعادة تحميل
