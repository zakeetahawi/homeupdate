### اختبار متقدم:

```bash
python manage.py shell

from accounts.models import User
from accounts.theme_customization import ThemeCustomization

user = User.objects.first()
tc = ThemeCustomization.objects.filter(user=user).first()

if tc:
    print("✅ التخصيصات محفوظة")
    print("Primary:", tc.primary_color)
    print("CSS Vars:", tc.get_css_variables())
else:
    print("❌ لا توجد تخصيصات")
```
